JAWE Just Another Web Editor JAWE works by using custom elements to overlay a div element atop of a textarea element, then uses Markdot 0.2 syntax highlighting. This allows JAWE to use the browsers built in spell check. Because JAWE uses custom elements and some other new features, it requires a very recent version of Chrome to function. Pollyfills will be provided in the final version of JAWE. JAWE superseeds Virtuoso (http://github.com/benjamingwynn/virtuoso) Right now JAWE only supports one current document, and stores said document in localStorage. In the future multiple document support is planned. To empty the document and return to this default, do the following in the console: localStorage.clear() Written by Benjamin Gwynn - xenxier@xenxier.com
